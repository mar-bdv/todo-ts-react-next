# To-Do приложение
### - Реализовала Drag & Drop для перемещения задач в списке, улучшив UX взаимодействия
### - Реализованы фильтры (Все/Активные/Выполненные)
### - Есть возможность редактировать задачу, а также присваивать ей категорию
## Технологии:
<ul>
